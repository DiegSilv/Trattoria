Seja Bem-vindo(a) ao Processo de Compra de Ingressos para a Trattoria 2023!

Para garantir uma experiência tranquila, siga as orientações abaixo:

1. Preencha os dados do comprador novamente:

Os dados inseridos na seção anterior do comprador não são utilizados para o preenchimento do ingresso. caso deseja comprar um ingresso para o comprador, insira os dados novamente no campo do ingresso

2. Escolha o tipo de ingresso:

Clique nos botões "Ingresso Adulto" ou "Ingresso Criança" para adicionar ingressos à sua compra. Os valores referentes a cada ingresso são exibidos em cada botão

3. Preencha os dados para cada ingresso adicionado:

Nome: Insira o nome completo do titular do ingresso no campo "Nome"

CPF: Digite o número do CPF no formato XXX.XXX.XXX-XX (para crianças e bebês esse campo é opcional)

Data de Nascimento: Forneça a data de nascimento da criança ou bebê no formato apropriado

4. Restrição Alimentar (opcional):

Caso o titular do ingresso tenha alguma restrição alimentar, você pode especificá-la no campo "Possui alguma restrição alimentar?"

5. Conhece Alguém no Evento:

Se o titular do ingresso conhece alguém no evento, selecione a opção correspondente. Escolha entre "Aluno Que Faz Técnico", "Aluno Que Não Faz Técnico", "Sou Professor" ou "Não". Para cada ingresso adicionado, a seleção do aluno deve ser feita novamente. Essa seção serve para a organização das mesas, de forma que os ingressos com o mesmo aluno de referência, sentarão juntos

6. Excluindo Ingressos:

Se você precisar excluir um ingresso, clique no botão "Excluir" correspondente

7. Dados Inválidos:

Se algum dado estiver incorreto ou faltando, os campos com problemas serão destacados em vermelho. Certifique-se de preencher corretamente esses campos antes de prosseguir

8. Idade da Criança:

Lembre-se de que a data de nascimento fornecida para ingressos de criança deve corresponder à faixa etária de 4 a 12 anos. Caso contrário, você será notificado e terá a opção de converter o ingresso para adulto
